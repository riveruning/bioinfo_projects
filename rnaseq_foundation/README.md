# RNA-seq Foundation

This project records my step-by-step learning process for RNA-seq analysis.

## Goal

The goal of this project is to understand how raw RNA-seq data can be processed into gene expression results and biological interpretation.

In this project, I will learn how to move from raw sequencing reads to biological conclusions through a complete RNA-seq workflow.

## Core Question

How can RNA-seq data be transformed from raw reads into gene expression results?

## Project Structure

- `metadata/`: sample information and experimental design
- `data/`: raw data and processed data
- `scripts/`: analysis scripts and command-line workflows
- `results/`: analysis outputs, tables, and figures
- `notes/`: daily learning notes
- `literature/`: paper reading notes and references

## Planned Workflow

The basic RNA-seq workflow is:

1. Understand the biological question
2. Prepare sample metadata
3. Inspect FASTQ files
4. Perform quality control
5. Quantify gene or transcript expression
6. Build a count matrix
7. Perform differential expression analysis
8. Visualize results
9. Interpret biological meaning

## Current Stage

Day 1: Understanding what an RNA-seq project analyzes.

## Notes

This is a learning project. The focus is not only on running commands, but also on understanding why each step is needed.
